# RubberDock
A simple dock system made for React. Inspired by GoldenLayout.

# Overview
This lightweight framework allows you to use React to create a grid-based docking system.

# Features
* Fast
* Styleable
* Simple Layout (layout, column, row, stack, and item)

# Topology
* Layout
  * Row
  * Column
    * H. Stack
    * V. Stack
      * Item
        * Content
